## Airline Management System

Install the dependencies:

```bash
npm i
# or
yarn i
# or
pnpm i
# or
bun i
```

Create .env file:

```bash
TURSO_DATABASE_URL=""
TURSO_AUTH_TOKEN=""
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000)
